# 株式ポートフォリオ最適化システム - 自動実行ガイド

このガイドでは、株価予測からポートフォリオ最適化までを自動実行する方法を説明します。

## 📋 概要

自動実行スクリプトは以下の処理を順番に実行します：

1. **株価予測モデルの実行** (Python + LightGBM)
   - 119銘柄の株価データを読み込み
   - 14種類のテクニカル指標を計算
   - 5営業日後の収益率を予測
   - 予測結果を `data/prediction_results.csv` に保存

2. **ポートフォリオ最適化の実行** (IBM CPLEX)
   - 予測結果を読み込み
   - 制約条件を満たす最適なポートフォリオを計算
   - 結果を表示・保存

## 🔧 トラブルシューティング

### エラー: "Pythonが見つかりません"

**原因**: Pythonがインストールされていない、またはパスが正しくない

**解決方法**:
1. Pythonがインストールされているか確認
2. `run_optimization.bat` のPythonパスを修正
3. または、システムのPATH環境変数にPythonを追加

### エラー: "ModuleNotFoundError: No module named 'pandas'"

**原因**: 必要なPythonパッケージがインストールされていない

**解決方法**:
```bash
pip install pandas numpy lightgbm scikit-learn
```

### エラー: "CPLEXのコマンドラインツールが見つかりません"

**原因**: CPLEXがインストールされていない、またはパスが正しくない

**解決方法**:
1. IBM CPLEX Optimization Studioをインストール
2. `run_optimization.py` の `cplex_paths` を確認・修正
3. または、手動でCPLEX IDEから実行：
   - CPLEX IDEを起動
   - `optimization/portfolio.mod` を開く
   - 右クリック → "Run This"

### エラー: "予測結果ファイルが見つかりません"

**原因**: 株価予測モデルの実行に失敗している

**解決方法**:
1. `data/sampledata.csv` が存在するか確認
2. 手動で予測モデルを実行してエラーを確認：
   ```bash
   cd models
   python stock_prediction.py
   ```

## 📝 注意事項

### 実行時間
- 株価予測: 約30秒～1分
- ポートフォリオ最適化: 約5～10秒
- 合計: 約1～2分

### データの更新
- 株価データ (`data/sampledata.csv`) を更新すると、新しいデータで予測が実行されます
- 予測結果は毎回上書きされます

### 結果の保存
- 最適化結果は `results/` フォルダにタイムスタンプ付きで保存されます
- 過去の結果も保持されるため、履歴を確認できます

## 🔄 定期実行の設定（オプション）

Windowsタスクスケジューラを使用して、定期的に自動実行することもできます。

### 設定手順

1. **タスクスケジューラを開く**
   - スタートメニュー → "タスクスケジューラ" で検索

2. **基本タスクの作成**
   - 右側の「基本タスクの作成」をクリック
   - 名前: "株式ポートフォリオ最適化"
   - 説明: "毎日自動実行"

3. **トリガーの設定**
   - 毎日、または毎週など、希望の頻度を選択
   - 実行時刻を設定（例: 毎日 9:00）

4. **操作の設定**
   - 「プログラムの開始」を選択
   - プログラム/スクリプト: `C:\Bob\20260420\run_optimization.bat`
   - 開始: `C:\Bob\20260420`

5. **完了**
   - 設定を確認して完了
